# @lumenflow/core

Core WU lifecycle tools for the LumenFlow workflow framework.

## Installation

```bash
pnpm add @lumenflow/core
```

## Usage

```typescript
import { VERSION } from '@lumenflow/core';

console.log(`LumenFlow Core v${VERSION}`);
```

## Status

This package is under active development. The v1 foundation provides:

- Modern tooling (Node 22, ESLint 9, TypeScript 5.7, Vitest 4)
- Hexagonal architecture boundaries
- Security and code quality linting
