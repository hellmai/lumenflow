The **Agent Runtime Pack** provides governed model turns and `agent-session` orchestration on top
of the [kernel](/kernel/runtime). It keeps policy enforcement, scope enforcement, evidence, and
sandboxing in the kernel while adding a pack-owned contract for agent execution.

<Aside type="note">
  The kernel remains the enforcement boundary. The pack adds agent-session semantics, provider
  adapter normalization, and workflow state.
</Aside>
