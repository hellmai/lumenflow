## Upgrade Guide

To upgrade to the latest version:

```bash
pnpm lumenflow:upgrade --latest
```

See the [Upgrade Guide](/getting-started/upgrade) for detailed instructions.

## Backwards Compatibility

LumenFlow follows [Semantic Versioning](https://semver.org/):

- **Major versions** (X.0.0) may include breaking changes
- **Minor versions** (0.X.0) add features, backwards compatible
- **Patch versions** (0.0.X) fix bugs, backwards compatible

When upgrading across major versions, always check the release notes or changelog entry for
migration steps.
