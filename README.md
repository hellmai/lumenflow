# HellmAI Operating System

LumenFlow v2.0 + COS v1.3 frameworks for multi-product reuse.

## Quick Start

**New to these frameworks?** Follow the [5-step adoption guide](ADOPTION.md) to integrate LumenFlow and COS into your product.

**Time estimate:** 1-2 hours for initial setup

### What's Included

- **LumenFlow v2.0:** Workflow framework with worktree discipline, WIP=1, and TDD-first approach
- **COS v1.3:** Company Operating System with governance rules and evidence-based compliance
- **Templates:** Configuration files for `.lumenflow.config.yaml`, project rules, and WU schema v2.0
- **Documentation:** Complete framework specifications and implementation guide

### Prerequisites

- Git repository with main branch
- Node.js project with package.json (or equivalent for your stack)
- Basic familiarity with YAML and git workflows

### Get Started

```bash
# 1. Clone this repo
git clone git@github.com:hellmai/os.git /tmp/hellmai-os

# 2. Follow the adoption guide
open /tmp/hellmai-os/ADOPTION.md
```

See **[ADOPTION.md](ADOPTION.md)** for complete step-by-step instructions.
